# 💉 Inyección ciega con retrasos temporales

## 📄 Descripción del laboratorio

La funcionalidad **Submit feedback** ejecuta un comando del sistema operativo que incluye varios parámetros controlados por el usuario.

Aunque el comando se ejecuta en el servidor, la salida no se refleja en la respuesta HTTP, lo que convierte el caso en una **OS Command Injection ciega**.

🎯 **Objetivo del laboratorio:**

* Provocar un retraso de **10 segundos**
* Confirmar la ejecución del comando inyectado
* Resolver el laboratorio

 

## 📚 Teoría

En una **inyección de comandos ciega**, no podemos ver directamente la salida del comando.

La confirmación se basa en **efectos colaterales observables**, como:

* ⏱️ Retrasos temporales (`sleep`, `ping`, `timeout`)
* 🔁 Cambios de comportamiento de la aplicación
* 🌐 Interacciones externas (DNS/HTTP out-of-band)

En este laboratorio usamos un **payload basado en tiempo**:

```bash
sleep 10
```

Alternativas comunes:

```bash
ping -c 10 127.0.0.1
```

```
timeout 10
```

Si la respuesta tarda aproximadamente **10 segundos**, el comando se ha ejecutado.

 

## 📝 Práctica

### 1️⃣ Interceptar el feedback

Accedemos a **Submit feedback**.

Enviamos un formulario de prueba e interceptamos la petición **POST** con **Burp Suite**.

La petición contiene varios parámetros, por ejemplo:

* `name`
* `email`
* `subject`
* `message`

 

### 2️⃣ Identificación del parámetro vulnerable

Como no sabemos qué parámetro se concatena al comando del sistema, probamos inyectar el payload en cada uno y observamos el tiempo de respuesta.

Payload utilizado:

```bash
;sleep 10;
```

Alternativa válida:

```bash
;ping -c 10 127.0.0.1;
```

 

### 3️⃣ Ejecución del payload

Al inyectar el payload en el parámetro `email`:

```
email=test@test.com; sleep 10
```

Enviamos la petición modificada.

### 4️⃣ Resultado final

La respuesta tarda aproximadamente **10 segundos** en completarse y el retraso es consistente y reproducible.

Esto confirma:

* **Ejecución de comandos en el servidor**
* **Inyección de comandos ciega**
